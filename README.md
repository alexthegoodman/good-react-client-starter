# Good React Client Starter

Welcome

## Getting Started

`heroku login`
`heroku buildpacks:set https://github.com/heroku/heroku-buildpack-static.git`
`heroku plugins:install heroku-cli-static`

`yarn`
`yarn dev:client`

## Deploy

`yarn build:client`
<!-- `yarn heroku:deploy` -->
`heroku static:deploy`
